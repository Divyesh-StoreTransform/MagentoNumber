# Internation Mobile Number input validation for Magento 2 using Google jQuery plugin
This module for Magento 2 is based on a jQuery Plugin for entering and validating Mobile Numbers. The utilities script (build/js/utils.js) is a custom build of Google's <a href="https://github.com/googlei18n/libphonenumber">libphonenumber</a> plugin.

Integration to Magento 2 a jQuery plugin for entering and validating Mobile Numbers. It adds a flag dropdown to Mobile Number fields, detects the user's country, displays a relevant placeholder and provides formatting/validation methods.
